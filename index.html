My Learning Progress: Git & GitHub

Here is a summary of what I have learned so far about Git, GitHub, GitHub Desktop, and how to host projects on GitHub!

1. What is Git?

  • Git is a distributed version control system.
  • It helps track changes in code or files over time.
  • Git allows multiple people to work on the same project without interfering with each other.
  • It keeps the history of all changes and lets you revert to previous versions when needed.

2. What is GitHub?

  • GitHub is a web-based platform that hosts Git repositories online.
  • It lets you collaborate with others, manage projects, and share your code publicly or privately.
  • GitHub adds features like issues, pull requests, and project boards to help teams work together.

3. What is GitHub Desktop?

  • GitHub Desktop is a graphical user interface (GUI) application for Git and GitHub.
  • It makes it easier to manage repositories, commits, branches, and pushes/pulls without using the command line.
  • You can visualize changes, resolve conflicts, and sync with GitHub with just a few clicks.

4. How to Create a GitHub Account and Repository

  1. Create a GitHub Account:
    • Go to https://github.com/
    • Click Sign up and follow the instructions.
  2. Create a Repository:
    • After logging in, click the + icon (top right) → New repository.
    • Enter a repository name, description, choose public/private, and click Create repository.

5. What is a Commit?

  • A commit is a snapshot of your project at a certain point in time.
  • Commits record what files changed, who made the changes, and a message explaining the changes.
  • Each commit has a unique ID (hash) and builds the project's history.

6. How to Host a Project on GitHub (using GitHub Desktop)

  1. Clone or Create a Repository:
    • Open GitHub Desktop and sign in.
    • Click File → New repository to create a new one, or Clone repository to work on an existing one.
  2. Add Your Project Files:
    • Copy your files into the repository folder on your computer.
  3. Commit Your Changes:
    • In GitHub Desktop, you'll see the changed files.
    • Add a commit message describing your changes.
    • Click Commit to main (or your branch name).
  4. Push to GitHub:
    • Click Push origin to upload your changes to your GitHub repository online.
  5. View Your Project Online:
    • Go to your repository page on GitHub to see your project and files.

Summary

I have learned the basics of Git and GitHub, how to use GitHub Desktop, and the steps to create an account, make a repository, commit changes, and host my project on GitHub!
